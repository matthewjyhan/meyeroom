How Does This Work?

1. Pick a temporary username (2-20 characters).

2. Join any public room or create a private room with a password.

3. Send regular messages to everyone in the room.

4. Mention a user with @username. Everyone sees the mention styled, and the mentioned person sees the whole message highlighted.

5. Send a whisper with /w @username "your message". Only you and that user can see it.

6. Change your username color with /color "color". Pick from red, orange, yellow, green, blue, or violet.

Tip: Keep messages respectful and avoid spamming to stay connected.

About Meyeroom

Meyeroom is a nostalgic, web-based chat application inspired by the classic AOL chat rooms of the 1990s. Enjoy the retro vibes!


Features

  • Public chat rooms for general discussion
  • Private password-protected rooms
  • User mentions and private whispers
  • Customizable username colors
  • No registration required

Our Name

Meyeroom is a play on the words "My" and "Room," using the homophone "eye" to represent a search for deeper human connection.
